# Java Data Structures Exercises

This repository contains my Java practice examples focused on basic data structures and collection usage.

## Topics Covered
- LinkedList operations
- Stack operations
- Queue operations
- Searching, removing, and displaying data
- Simple menu-based program flow

## Included Examples

### Student Scores with LinkedList
Demonstrates adding, removing, searching, and iterating through student scores using LinkedList.

### To-Do List Manager
A simple menu-driven program that manages tasks using LinkedList.

### Stack and Queue Basics
Shows basic stack and queue operations such as push, pop, peek, add, and poll.

## Purpose
I created this repository to organize my coursework exercises and improve my understanding of Java collections and basic data structures.
